以下为必须实现的命令行参数:

### Unix Usage:

**-c**:  Execute String Command

**-s**:  Force stdin as Command Source

**-i**:  Force Into Interactive Mode

**-** : Option Terminator (Stop Parsing Options)

### RDP Usage:

-h/--help : Output This Message and Exit

-ri/--run-ir <ir> : Execute Optimized Binary Format IR

-do/--debug-output : Output Every Debug Output to Stdout

-di/--dump-ir <source> : Optimize the Source and Save it's IR in Binary Format

-hi/--human-ir <source> : Optimize the Source and Save it's IR in Human-Readable Format
